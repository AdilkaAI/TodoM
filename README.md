# ToDoMini - Планировщик задач

Итоговый проект по HTML, CSS и JavaScript

## Страницы

- **Главная**  список задач, добавление, фильтры  
- **Статистика**  общая статистика и прогресс  
- **О проекте**  описание проекта  
- **Обратная связь**  форма с валидацией  

## Реализованный функционал

Добавление, удаление, завершение задач
Приоритеты (низкий / средний / высокий)
Фильтрация (все / активные / выполненные)
Сохранение в localStorage
Адаптивный дизайн (мобильные, планшет, десктоп)
Анимации появления задач
 Валидация формы обратной связи
Семантическая верстка
Модульная структура CSS (reset, base, layout, components)
