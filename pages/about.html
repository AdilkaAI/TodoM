О проекте ToDoMini

ToDoMini - современный и удобный планировщик задач, созданный как итоговая работа по HTML, CSS и JavaScript.

Цель проекта

Продемонстрировать владение основами HTML/CSS/JS: семантическая верстка, адаптивность, DOM, события, формы, валидация, localStorage.

Что реализовано

  • Добавление задач с выбором приоритета
  • Отметка выполненных задач
  • Фильтры: Все / Активные / Выполненные
  • Сохранение в localStorage
  • Подробная статистика + круг прогресса
  • Форма обратной связи с валидацией
  • Адаптивный дизайн (мобильный, планшет, десктоп)
  • Гамбургер-меню на мобильных устройствах
  • Плавные анимации и hover-эффекты

Технологии

HTML5
CSS3 (Flexbox + Grid)
JavaScript ES6+
localStorage
